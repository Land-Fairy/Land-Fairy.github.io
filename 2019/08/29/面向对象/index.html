JAVA 面向对象

1. 匿名对象

没有名字的对象，其实就是 定义对象的简写形式

  • 仅需要对方法做一次调用时使用
  • 可以作为实际参数来进行传递

2. 封装

隐藏对象的属性和实现细节，仅对外提供公共的访问方式

3. 构造函数

  • 特点
  1. 函数名与类名相同
  2. 不用定义返回值类型
  3. 不可以写 return 语句
  • 注意:
  1. 多个构造函数是以重载的形式存在的
  2. 自定义了构造函数后，系统不会生成默认的构造函数

4. 构造代码块

5. this

代表所在函数所属对象的引用:

那个对象调用该方法，this 就代表该对象

6. 静态代码块

  • 特点
  1. 在类中方法外出现，并加上 static 修饰
  2. 用于给类进行初始化，并在加载的时候运行
  3. 只会执行一次
  4. 一般用于加载驱动
